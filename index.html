HTML

HTML es el lenguaje estándar utilizado para crear y estructurar las páginas web. Utiliza un sistema de etiquetas para definir elementos como texto, imágenes, enlaces, formularios y otros contenidos que aparecen en un sitio web. HTML le indica al navegador qué contenido debe mostrar, pero no controla la apariencia ni el estilo de ese contenido.

CSS

CSS es un lenguaje utilizado para definir la presentación, el diseño y la apariencia de los contenidos escritos en HTML. Controla cómo se muestran los elementos, como sus colores, fuentes, espaciado y posicionamiento. CSS permite separar la estructura del contenido (HTML) de su apariencia visual.
